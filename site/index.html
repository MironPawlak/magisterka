About the Project

I am working on a predictive and recommendation system for the game League of Legends, which will have two main functionalities:

  1. Predicting your chance to win and recommend characater to increase winning chances, based on historical data
  2. Provide statistics including winratio, pickratio and counters for specific champion

Technical Details:

  • The system uses machine learning (binary classification for match outcomes.
  • Data is sourced from the official Riot Games API
  • Analyzed factors: match outcomes, player stats, champion data, and actions.

This project is fully open source. You can view the code for both the client and server on GitHub.
